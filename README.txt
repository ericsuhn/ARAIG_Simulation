ARAIG Program Commands-

Type n and press enter to execute the next Task.
Type l and press enter to see list of tasks to be run.
Type k and press enter to list tasks that have been completed.
Type x and press enter to exit the program.


Command line arguments:
milestone.exe SampleProfileConfiguration.txt StimulationConfig.txt TaskConfiguration.txt outputFilename
